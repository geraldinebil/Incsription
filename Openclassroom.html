Les langages du développeur web.

Frontend, backend

Qu’est-ce que ça veut dire ?

Dans la programmation informatique il existe différents langage de programmation et de jargon technique. Il y a le frontend et le backend. Le backend est la partie invisible de l'iceberg. On utilise les langages de programmation comme Java, Python, PHP, C++, MySQL etc...On se sert de ces langages de programmation pour faire des bases de données. Le frontend lui est la face visible de l'iceberg. cela sert à créer des pages de navigations internet. Pour ce faire on utilise les langages de programmation comme HTML, CSS et JavaScript. C'est ce que nous verrons dans ces pages.

Le HTML

Qu'est ce que le HTML?

Le HTML est un langage de code qui permet de faire apparaître tout ce que l'on veut voir sur un site Web. Du texte, des images, des audios ainsi que des vidéos.Pour ce faire, tout ce que vous voulez voir apparaître doivent se trouver dans une balise ouvrante comme celle- ci: < > et une balise fermante comme celle-ci: .(Cliquez sur le logo HTML pour plus d'explications.)

Le CSS

Qu'est ce que le CSS?

Le CSS est la mise en forme de tout ce que l'on a mis dans le HTML. La taille du texte, la couleur du texte, l'épaisseur du texte, la taille des fichiers (images, vidéos). On peut également placer des éléments à droite, au milieu ou à gauche. Il est également possible d'encadrer du texte de mettre un fond dans le body etc... Pour créer ses mises en formes il suffit de créer une page CSS qui sera intégrer grâce un link rel dans son HTML, ensuite on onvrira des balises comme celle-ci { } pour pouvoir mettre en forme nos éléments.(Cliquez sur le logo CSS pour plus d'explications)

Le JS

Qu'est ce que le JS (JavaScript)?

Le JavaScript est un langage de programmation qui permet de créer des pages plus complexes et dynamiques. On peut avec le JS animer des images, faire des carousels d'images. Ne pas confondre avec le Java qui lui est un langage de programmation qui n'a pas pour but d'apparaître sur le site Web. Ici on ne parle plus de balise mais de variables. (Cliquez sur le logo JS pour plus d'explications)
